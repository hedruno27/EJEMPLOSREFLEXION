# Ejemplos de Reflexión en OOP

Este proyecto contiene ejemplos básicos de reflexión en tres lenguajes orientados a objetos:

- Java
- Python
- Ruby

## Archivos incluidos

- [ReflexionJava.java](ReflexionJava.java): ejemplo de reflexión en Java.
- [reflexion_python.py](reflexion_python.py): ejemplo equivalente en Python.
- [reflexion_ruby.rb](reflexion_ruby.rb): ejemplo equivalente en Ruby.
- [GUIA_PRACTICA_EJECUCION.md](GUIA_PRACTICA_EJECUCION.md): guía de apoyo para la tarea.
- [Tarea_03_Solucion_Completa.md](Tarea_03_Solucion_Completa.md): solución completa de la actividad.

## Cómo ejecutar

### Java
```bash
javac ReflexionJava.java
java ReflexionJava
```

### Python
```bash
python reflexion_python.py
```

### Ruby
```bash
ruby reflexion_ruby.rb
```

## Objetivo

Mostrar cómo se puede inspeccionar, invocar y modificar comportamiento de objetos de forma dinámica mediante reflexión, comparando el enfoque entre lenguajes.
